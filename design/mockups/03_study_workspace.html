Exercise 3: Haskell List Comprehension

Implement a function filterPrimeSquares that takes a list of integers and returns squares of prime numbers found within a given range $[a, b]$.

Mathematical Definition:

$$S = \{ x^2 \mid x \in L, \text{is_prime}(x), a \le x \le b \}$$

Constraints:

  • $L$ is an infinite list of integers.
  • Use recursion only for the primality test.
  • List comprehension must be used for the final mapping.
Consider using a helper function isPrime :: Integer -> Bool. A simple optimization is to check divisibility up to $\lfloor \sqrt{n} \rfloor$.

module Main where

-- Filter and square primes in range
filterPrimeSquares :: [Integer] -> Integer -> Integer -> [Integer]
filterPrimeSquares list a b = [x*x | x <- list, a <= x, x <= b, isPrime x]

isPrime :: Integer -> Bool
isPrime n
| n < 2 = False
| otherwise = null [x | x <- [2..floor (sqrt (fromIntegral n))], n `rem` x == 0]

main :: IO ()
main = print $ filterPrimeSquares [1..100] 10 50
